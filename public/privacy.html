Privacy Policy

Users personal information


The application stores users e-mail id and name in the applications data base. The users password is not stored directly in the database. To protect the theft of users login credentials, the application stores the users passwords hash only.

The personal information stored in the application database is not shared with any other third party.
Users can request for their accounts and all the personal information stored with the account to be deleted from the database forever by sending a request from their registered e-mail id to the following e-mail address: deepaksinghjadon@gmail.com
